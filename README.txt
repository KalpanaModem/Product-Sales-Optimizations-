Project Title:
Retail Product Recommendation and Sales Optimization Using Ant Colony Optimization, Genetic Algorithm, and Swarm Intelligence

Algorithms Used:
1. Ant Colony Optimization
2. Genetic Algorithm
3. Particle Swarm Optimization

Project Purpose:
To recommend products, generate profitable bundles, and optimize recommendation scores in a retail market.

How It Works:
ACO finds related products.
GA finds the best product bundle.
PSO optimizes recommendation weights.
